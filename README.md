 # Full Stack Job Listing Application

This repository contains my Full Stack Assignment, consisting of a frontend and backend application built with modern web technologies.

## Live Links

Frontend: https://mployee-assignment-frontend.vercel.app

Backend: https://mployee-assignment-backend.onrender.com

Frontend repo: https://github.com/kushagra2304/mployee-assignment-frontend.git

Backend  repo: https://github.com/kushagra2304/mployee-assignment-backend.git

## Tech Stack

Frontend: React, TypeScript, Axios, Tailwind CSS

Backend: Node.js, Express

Database: MongoDB (MongoDB Atlas)

## Features

Job listing with detailed job view

Location-based job search

Pagination for efficient handling of large data

Optimized REST APIs

Fully deployed frontend and backend

## Architecture

The frontend (React + TypeScript) communicates with the backend via REST APIs.
The backend (Node.js + Express) uses MongoDB Atlas for cloud data storage, with pagination implemented for better performance and scalability.

## Run Locally
# Backend
cd backend
npm install
npm run dev

# Frontend
cd frontend
npm install
npm run dev

## NOTES : To run it locally a .env file is also required which contains MONGO_URI link.
